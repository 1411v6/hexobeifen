# JavaScriptES6

## ES6箭头函数:

```
例:
let fn = val => val;
此语法等同于:
let fn = function(val){
    return val;
}
```

```
箭头函数:
* 简化函数的声明;
* 参数超过1个时必须加();
* 函数体超过1行时,必须加{},并自己写return.
```

```
箭头函数的作用:
* 简化回调函数;
* 箭头函数中的this是函数声明时所在的对象.
```
